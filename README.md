# Evolution Arena

Evolution Arena is a 2D survivor-like where you play as an evolving outbreak in a hostile biolab.

## Core Loop
- Move with `WASD`
- Defeat enemies and absorb biomass (XP)
- Level up and choose mutations or stats
- Survive timed containment events
- Defeat the final boss to win

## Current Build Highlights
- 3 variants: Lytic, Pandemic, Parasitic
- Mutation and stat choice system
- Event rewards + synergies
- Mid boss and final boss flow
- Genome Cache pods: random timed breakables that grant stat-only choices

## Controls
- `WASD`: move
- `1/2/3` or click: select level-up options
- `Esc`: pause

## Run
Open in Godot 4.6 and run `scenes/main_menu.tscn`.
